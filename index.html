What is Athena?

Athena is a fully asynchronous, non-blocking, high performance, stateless, transparent sharding supported, protocol layer database proxy.

Fully asynchronous

Both transportation from client to athena and that from athena to db are asynchronous, the proxy will never block your processes.

High performance

Athena support ELEME's all database in production for 4 years,and never cause an online accident which is higher than P4.

Stateless

Athena is a stateless proxy, you can install and run it anytime and anywhere.

Transparent sharding supported

Athena support sharding transparently.
